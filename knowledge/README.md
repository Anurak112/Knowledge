# Knowledge

This directory stores structured knowledge bases and domain-specific information that Agents can access. This is useful for RAG (Retrieval-Augmented Generation) or context injection.

## Structure

```
knowledge/
└── domains/
    └── domain-name.md    # Markdown file containing domain knowledge
```

## Format

Knowledge files should be well-structured Markdown to facilitate easy parsing and retrieval.

```markdown
# Domain Name

## Key Concepts
- Concept 1: Definition
- Concept 2: Definition

## Best Practices
1. Practice A
2. Practice B
```
