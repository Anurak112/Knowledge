# Settings

This directory contains configuration files for the global behavior of the AI assistant, as well as user profiles and themes.

## Structure

-   **`global.yaml`**: System-wide defaults.
-   **`profiles/`**: User-specific configuration overrides.
-   **`themes/`**: UI theme definitions (colors, fonts).

## Format

Settings are typically stored in YAML files for readability.

```yaml
language: en-US
theme: dark
editor_preference: vscode
```
