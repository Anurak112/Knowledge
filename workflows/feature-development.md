# Feature Development Workflow / ขั้นตอนการพัฒนาฟีเจอร์ใหม่

1. **Requirement Analysis**: Understand the user need.
2. **Design**: Create a design document or plan.
3. **Implementation**: Write the code (TDD preferred).
4. **Testing**: Unit tests and integration tests.
5. **Code Review**: Peer review.
6. **Merge & Deploy**: Merge to main and auto-deploy.
