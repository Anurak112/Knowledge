# Workflows

Workflows define structured processes or Standard Operating Procedures (SOPs) for the agent to follow. They are linear or branching sequences of steps to achieve a specific goal.

## Structure

Workflows are generally single markdown files located directly in this directory.

## Format

```markdown
---
description: Short title of the workflow
tags: [category, type]
---

# Workflow Title

1.  **Step One**: Description of action.
2.  **Step Two**: Description of action.
    -   Sub-step detail.
3.  **Step Three**: Verification.
```
