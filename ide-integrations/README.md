# IDE Integrations

This directory contains configuration files for integrating the Knowledge Repository with various IDEs.

## Supported IDEs

-   **VS Code**: `vscode/`
-   **Cursor**: `cursor/`
-   **Windsurf**: `windsurf/`
-   **Gemini Code Assist**: `gemini/`

## Usage

Symlink or copy these settings to your IDE's configuration directory.
