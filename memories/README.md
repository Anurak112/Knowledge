# Memories

This directory is used for long-term storage of agent memories. This includes conversation logs, context snapshots, and summarized information about the user and project.

## Categories

-   **`conversations/`**: Full logs of past interactions.
-   **`context/`**: Snapshots of the working context at specific times.
-   **`summaries/`**: Condensed insights and facts "learned" by the agent.

## Format

Memories are often stored as JSON or Markdown files, indexed by ID or date.
